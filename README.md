## Local development

### Required software

* Ruby version 2.5.2

### Database setup

```shell
bin/rails db:setup
```

### Run Specs (Tests)

```shell
bundle

### Start local servers

 ```shell
bundle exec foreman start
=> You should access http://localhost:5000/
 or
 (on different tabs)
bin/rails server
bin/webpack-dev-server
```
